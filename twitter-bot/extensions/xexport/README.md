# xExport — Export Twitter Followers & Following

> **Archived Chrome extension — not my code.**

The easiest way to export Twitter/X followers and following lists.

- **Version:** 1.0.4
- **Manifest:** V3
- **Chrome Web Store:** [xExport](https://chromewebstore.google.com/detail/xexport-export-twitter-fo/cppccmpggnomnbajdchpjahagpockdmc)
- **Creator website:** [tweetsearcher.com](https://tweetsearcher.com/)

## Install (Sideload)

1. Open Chrome and go to `chrome://extensions`
2. Enable **Developer mode** (toggle in the top-right corner)
3. Click **Load unpacked**
4. Select this `xexport/` folder
5. The extension icon will appear in your toolbar — navigate to any X/Twitter profile to use it
