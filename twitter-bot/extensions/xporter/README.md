# XPorter — Free X/Twitter Export

> **Archived Chrome extension — not my code.**

Export posts, followers & following from any X (Twitter) profile to CSV, JSON, or Excel. Free, unlimited, 100% private. Supports 14 languages.

- **Version:** 1.0.10
- **Manifest:** V3
- **Chrome Web Store:** [XPorter](https://chromewebstore.google.com/detail/xporter-free-x-twitter-ex/jghmghialodmkmbcpfnhkgllkmjafmja)

## Install (Sideload)

1. Open Chrome and go to `chrome://extensions`
2. Enable **Developer mode** (toggle in the top-right corner)
3. Click **Load unpacked**
4. Select this `xporter/` folder
5. The extension icon will appear in your toolbar — navigate to any X/Twitter profile to use it
